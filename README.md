# kala-rails

jquery.kala for Rails 3.1+ asset pipeline

### Installation

In your Gemfile, add this line:

    gem "kala-rails"

Then, run `bundle install`.

In your app/assets/javascripts/application.js, add this line:

    //= require jquery.kala

You're done!


### Use

See http://github.com/archit/kala for usage info.
